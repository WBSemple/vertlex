# Vertlex 🔗📚

## Requirements

* Node 20 - `nvm i 20`
* Clojure - [install](https://clojure.org/guides/install_clojure)

### Optional

* Just - [install](https://github.com/casey/just?tab=readme-ov-file#installation)  
  (to run commands in justfile)
* React Developer Tools - [doc](https://react.dev/learn/react-developer-tools)  
  (to enable fast refresh)
* Dataspex - [firefox](https://addons.mozilla.org/en-US/firefox/addon/dataspex) / [chrome](https://chromewebstore.google.com/detail/dataspex/blgomkhaagnapapellmdfelmohbalneo)  
  (to inspect app db in dev tools)

## Run

```
just run
```
